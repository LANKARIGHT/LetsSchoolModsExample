# 项目的标题和简要描述
 这是一个学园构想家的模组开发案例仓库，你可以通过参考这些案例来制作模组。

## 安装说明
    1.安装《学园构想家》
    2.复制内容到C:\Users\UserName\AppData\LocalLow\Pathea Games\LetsSchool\LocalMods

## 使用说明
   
    2.进入游戏
    3.在主菜单选择模组
    4.找到并加载案例模组
    5.进入游戏查看模组内容
